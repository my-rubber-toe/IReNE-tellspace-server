Authentication module¶

Authentication Module: authentication.py

Holds the endpoints that perform authentication operations. These endpoints are responsible for the generation, refreshing and revoking of access tokens.

authentication.auth_logout()¶

Revoke access token. Add access tokens to blacklist by extracting the JSON Token identifier.

ApiResult

the response object containing a JSON Object with acknowledgment message.

TellSpaceAuthError

Exception class for authentication errors.

authentication.auth_me()¶

” Get the collaborator information from the database. (First Name, Last Name, Email and id)

ApiResult

the response object containing a JSON Object with the collaborators first name, last name, email and id.

TellSpaceAuthError

Exception class for authentication errors.

authentication.auth_refresh()¶

Return a new access_token given a valid refresh token.

ApiResult

the response object containing a JSON Object with the new access_token

TellSpaceAuthError

Exception class for authentication errors.

authentication.bp = <flask.blueprints.Blueprint object>¶

Instance of a Flask “Blueprint” class to implement a custom endpoint groups.

authentication.check_if_token_in_blacklist(decrypted_token)¶

Verifies if a token has been blacklisted if the time-to-live cache by extracting the JSON token identifier.

decrypted_token

the token string to be evaluated.

entry

the key of the blacklist set which points to the blacklisted token.

False

boolean value if a token is not found in the blacklist.

authentication.get_tokens(google_token: str)¶

Verify if the given param string is a valid Google idToken issued by Google and made with the projects credentials. If the user is NOT banned and is approved, proceed with token generation.

google_token

the JWT Google generated token to be evaluated.

ApiResult

the response object containing a JSON Object with two tokens: access_token and refresh_token, respectively.

TellSpaceAuthError

error exception class for authentication related errors.

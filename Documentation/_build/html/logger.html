Logger Module: logger.py¶

This module holds classes responsible for performing standarized log operations.Can be called by any other module to log events.

Class to perform logging operations for info and error logs.

log_error(message)¶

Log messages to the app.log file with ERROR level. If used in an “except” block it will log the whole stack trace error.

message

message string to log
